- Code daily in C, C++, and Go.

## Structure
- **YYYY/MM-DD/**: Daily challenges (e.g., `2025/06-14`).
  - Subfolders: `c/`, `cpp/`, `go/`.
  - `README.md`: Challenge details, learnings, run instructions.
- **resources/**: Algorithms, hacking tips, learning plan.
